## 数据可视化

数据往往是隐性的，尤其是当数据量大的时候很难感知，可视化可以帮我们很好的理解这些数据的结构，以及分析结果的呈现。

### 如何进行数据可视化?

- Python \* Matplotlib、Seaborn 等第三方库
- 第三方工具 \* 可以采用 微图、DataV、 Data GIF Maker 等
